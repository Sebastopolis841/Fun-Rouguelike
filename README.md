# Fun-Rouguelike

This is a text-based rouguelike game I am making in python, complete with RNG, loot tables, and no AI used.

## Status

This game is already playable, but I am still working on making it better. There is currently a skeleton room, a chest room and a stone golem room in the game.

## Download

Download the latest tested versio of this game either by clicking the link below or downloading the zip file above, and then clicking "download raw"

To play, you have to run main.py.
